# LLM Seed File Standard

## Purpose

LLM seed files store formula candidates generated by external AI agents (Gemini, GPT-4o, SciSpace) for use as warm-start seeds in the seeded-GP pipeline (Issue #27).

## File placement

```
data/llm_seeds/<disease>/<agent_slug>.csv
```

`data/llm_seeds/` is gitignored (covered by the top-level `data/` rule).

## CSV format

- Exactly one column named `expression`. No index column.
- Each row is one candidate formula string.
- Variables must use the `lab_<FEATURE>_last` naming convention.

Example valid row:
```
lab_NEUTpct_last / (lab_LYMpct_last)
```

## Variable name map

| Seed file variable | Pipeline feature |
|---|---|
| `lab_HCT_last` | `hct` |
| `lab_HGB_last` / `lab_HB_last` | `hgb` |
| `lab_MCH_last` | `mch` |
| `lab_MCHC_last` | `mchc` |
| `lab_MCV_last` | `mcv` |
| `lab_PLT_last` | `plt` |
| `lab_RBC_last` | `rbc` |
| `lab_RDW_last` | `rdw` |
| `lab_WBC_last` | `wbc` |
| `lab_NEUTpct_last` | `neut_pct` |
| `lab_LYMpct_last` | `lym_pct` |
| `lab_MONOpct_last` | `mono_pct` |
| `lab_EOS_pct_last` | `eos_pct` |
| `lab_BASO_pct_last` | `baso_pct` |

Translation is performed by `translate_seed_expression()` in `src/utils.py`.

## Agent slugs

| Agent | Slug |
|---|---|
| Gemini 2.5 Pro | `gemini_25_pro` |
| GPT-4o Deep Research | `gpt4o_deep_research` |
| SciSpace Agent | `scispace_agent` |
| Med-Gemma 4B (M4 internal) | `medgemma_4b` |
